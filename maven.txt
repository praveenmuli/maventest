Artifatcs:- is a final product application.
pom.xml:(Project Object Model) file- It will contains all the instructions for creating the projects
Maven is abuild tool.
It is used to work with Java based projects.
It is used to create the artifatcs such as ear, war and jar files.
ear-Enterprise Archive
War-WebArchive
jar-Java Archive

.class: Which can be executable
.jar: group of .class files
.war: it has both frontend and backend code.

mvn compile:- to compile the source code
mvn test:- to test the code
mvn package:- to create the new artifact(Project directory)
mvn install:- to create artifact(maven home dir .m2)
mvn clean package:- to perform entire goals
mvn clean:- To delete the artifact:- to delete artifact
